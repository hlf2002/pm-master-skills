# MBTI性格测试网站规格文档

## 1. 项目概述

- **项目名称**: MBTI.7373.com.cn 性格测试
- **项目类型**: 单页面Web应用
- **核心功能**: 基于MBTI理论的性格测试，包含28道题目，精确计算四个维度偏好，输出16种性格类型
- **目标用户**: 想要了解自己性格类型的用户

## 2. UI/UX 规格

### 布局结构

- **页面结构**:
  - 顶部导航栏（固定）
  - 欢迎/首页区域
  - 测试区域（题目展示）
  - 结果展示区域
  - 底部信息区

- **响应式断点**:
  - 移动端: < 768px
  - 平板: 768px - 1024px
  - 桌面: > 1024px

### 视觉设计

- **主色调**:
  - 背景色: #FFFFFF（纯白）
  - 主文字: #1a1a2e（深墨色）
  - 次要文字: #666666
  - 强调色: #4a90d9（蓝色）
  - 边框色: #e0e0e0

- **字体**:
  - 标题: "Noto Sans SC", "PingFang SC", sans-serif
  - 正文: "Noto Sans SC", "PingFang SC", sans-serif
  - 字号: 标题28px, 副标题20px, 正文16px, 小字14px

- **间距系统**:
  - 基础单位: 8px
  - 组件间距: 16px / 24px / 32px

- **视觉效果**:
  - 卡片阴影: 0 2px 12px rgba(0,0,0,0.08)
  - 按钮悬停: 轻微上移 + 阴影加深
  - 过渡动画: 0.3s ease

### 组件列表

1. **导航栏**: 简洁白色，左侧Logo，右侧说明链接
2. **首页卡片**: 介绍MBTI，测试说明，开始按钮
3. **进度条**: 显示当前题目进度（1/28）
4. **题目卡片**: 题目文字 + 两个选项（强迫选择）
5. **选项按钮**: 选中高亮，两选项并排
6. **结果卡片**: 四大维度雷达图 + 16种类型匹配度
7. **类型详情**: 所测类型的详细描述
8. **重新测试按钮**: 重新开始测试

## 3. 功能规格

### MBTI测试逻辑

**四个维度（每个维度2个偏好）**:
1. **E vs I** - 外向(Extraversion) / 内向(Introversion)
2. **S vs N** - 感觉(Sensing) / 直觉(Intuition)
3. **T vs F** - 思考(Thinking) / 情感(Feeling)
4. **J vs P** - 判断(Judging) / 知觉(Perceiving)

**题目设计**:
- 共28道题目（每维度7道）
- 采用Likert 5级量表设计，但最终转换为二元选择
- 选项设计为"强迫选择"形式，更符合MBTI传统

**16种性格类型**:
- ISTJ, ISFJ, INFJ, INTJ
- ISTP, ISFP, INFP, INTP
- ESTP, ESFP, ENFP, ENTP
- ESTJ, ESFJ, ENFJ, ENTJ

### 用户交互流程

1. 用户进入首页 → 查看介绍 → 点击"开始测试"
2. 逐题作答 → 每题选择A或B → 自动进入下一题
3. 28题完成后 → 显示结果
4. 结果页展示:
   - 四维度偏好条形图
   - 最终性格类型
   - 性格类型详细分析
   - 职业建议
5. 可选择重新测试

### 数据处理

- 本地计算，无需后端
- 使用localStorage保存进度（可选）
- 题目顺序可随机打乱（可选）
